⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/Ganesh052004/fashion-store-davangere.git
cd fashion-store-davangere
2️⃣ Install Python Packages
pip install flask flask-cors flask-bcrypt pymysql pyjwt
3️⃣ Start XAMPP

Start:

Apache
MySQL
4️⃣ Create Database

Open phpMyAdmin and create database:

CREATE DATABASE luxe_store;
5️⃣ Run SQL Tables

Create required tables:
users
products
cart
orders
order_items
6️⃣ Run Flask Server
python app.py

Server runs at:

http://127.0.0.1:5000
